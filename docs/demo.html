上传 zip 子集并分析（SEED 两协议）

上传一个 zip（包含 `EEF_features/` 与 `save_info/`），网页会在浏览器端完成推理，并按 SEED 两种协议给出指标。

目录规范： EEF_features/<subject>.mat， save_info/<subject>_<date>_<session>_save_info.csv（session 建议 1..4）。

说明：当前网页端支持 LR + MLP；首次点击“开始分析”时会加载 Pyodide（解析 `.mat`）与 JSZip（解压 zip），可能稍慢。 Pyodide 会依次尝试多个 CDN（jsDelivr / unpkg 等）；若某一 CDN 被网络屏蔽，会自动换源。仍失败时请换网络或使用可访问国际 CDN 的环境。
